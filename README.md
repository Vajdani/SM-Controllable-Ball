# This mod adds a ball that you can control and cause chaos with!

**Controls**
- Interact: Enter/Exit the ball
- WASD: Move around
- Space: Jump
- Left Click: Boost the ball forwards
- Right Click: Attach/Detach grappling hook
- Scroll Up: Zoom in
- Scroll Down: Zoom out
